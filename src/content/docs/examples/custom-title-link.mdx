By default, Starlight will use the [`site`](https://docs.astro.build/en/reference/configuration-reference/#site) value in `astro.config.mjs` to generate the link your site title points to, adding a language path to it for multilingual sites.

You can set any arbitrary link by using [route middleware](https://starlight.astro.build/guides/route-data/#customizing-route-data) and setting [`siteTitleHref`](https://starlight.astro.build/reference/route-data/#sitetitlehref) like in this example.
